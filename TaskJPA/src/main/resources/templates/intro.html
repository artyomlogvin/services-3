Пожалуйста, внимательно проверяйте введенные данные. Возможные причины ошибки: попытка добавить книгу с уже существующим id либо номером.

Некорректный ввод ограничения, помните значение принимаемое этим полем должно быть либо 1 либо 0

Пустые/нулевые/неподходящие по типизации данные и тд.

Однако, для того чтобы все работало должным образом, необходимо указать определенные параметры в файле application.properties и в функциях требующих подключения к БД, таковые имеются в файле IndexController, также немаловажно выставить соотвествующие зависимости в pom.xml.
